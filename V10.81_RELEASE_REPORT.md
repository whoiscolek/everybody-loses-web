# Everybody Loses v10.81 Release Report

## Production issue addressed

The Profile stats tab still contained placeholder “planned stats” copy even though the app now has enough settled bets to show real performance data. v10.81 replaces those placeholders with computed analytics based on the existing Firestore data already loaded by the client.

## Profile stats now implemented

The Stats page now computes and displays:

- Core settled decision totals: wins, losses, voids, win rate, net, total risked, and average stake.
- Sport splits from settled decisions.
- League splits from settled decisions.
- Pick splits, including individual UFC fight picks.
- Odds-tagged performance when a settled event or bet has non-placeholder odds attached.
- A clear count of settled decisions missing reliable odds, instead of hiding the feature behind placeholder text.
- Recent settled decisions with event, pick, league, and result.

The analytics are derived from `events`, `bets`, `matches`, and `ledgerEntries` in the browser state. No new backend dependency or data migration is required.

## Reliability guardrails added

- Added `src/profile-stats.js` as a pure, tested analytics module rather than embedding all logic directly in the renderer.
- Added regression tests for UFC bout stats, team-event stats, sport/league splits, pick splits, and odds-tagged performance.
- Added a repository-contract test proving the old planned-stat placeholder copy is gone.
